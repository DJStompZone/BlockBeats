## About Block Beats

**Block Beats** is a powerful custom audio mod that introduces a versatile block capable of playing audio files when powered by redstone. Inspired by the popular Java AudioPlayer mod, this block brings a unique feature set to Minecraft Bedrock, allowing players to customize their gameplay experience with personalized audio tracks.

### Key Features:

- **Redstone Integration:** The block can be toggled on and off using redstone power, seamlessly integrating with other redstone contraptions and circuits. Perfect for creating automated sound systems and musical contraptions.

- **Custom Audio Configuration:** Through an intuitive user interface, players can:
  - Select custom audio files to play
  - Configure track length and timing
  - Adjust pitch (0.1x to 2.0x)
  - Set audio load distance (0.01 to 100 blocks)
  - Enable/disable looping
  - Toggle pulse-based playback

- **Advanced Playback Control:**
  - Automatic looping based on configured track lengths
  - Pulse-based playback for precise control
  - Block locking system for protection
  - Persistent configuration storage

- **Security & Protection:**
  - Block locking system to prevent accidental configuration changes
  - Persistent storage of block configurations
  - Automatic cleanup of block data when destroyed

## Quick Start Guide

1. Place a Block Beats block in your world
2. Right-click to configure the block:
   - Enter the audio file name
   - Set track length (in minutes.seconds format)
   - Configure playback options
3. Power with redstone to play audio
4. Use a stick to lock/unlock the block

## Adding Custom Sounds

Want to add your own music or sound effects? Here's how:

1. **Prepare Your Sound File:**
   - Convert your audio to .ogg format (recommended) or .wav
   - Keep the file size reasonable (under 5MB)
   - Name it something simple (e.g., "ambient_forest.ogg")

2. **Add to Your World:**
   - Open your world's folder
   - Navigate to the "sounds" folder
   - Copy your sound file there

3. **Set Up the Block:**
   - Place a Block Beats block where you want the sound
   - Right-click to open the config menu
   - Enter your sound file name (e.g., "ambient_forest.ogg")
   - Set the track length (e.g., "3.45" for 3 minutes, 45 seconds)
   - Adjust volume and pitch as needed

4. **Test It Out:**
   - Power the block with redstone
   - Your sound should start playing!
   - Use a stick to lock the block and prevent accidental changes

*Pro Tip:* For background music, try setting the volume to around 30-50 and enabling looping!
